# KILN - Pottery Wheel Throwing & Clay Studio HTML Template

KILN is a premium, modern, and fully responsive HTML template designed specifically for pottery studios, ceramic workshops, and clay artisans. It features an organic, earthy design aesthetic that perfectly captures the tactile nature of working with clay.

## Features

- **14 Unique Pages**: Including 2 Home variations, About, Services, Contact, Booking, Gallery, Blog, and bespoke Dashboards.
- **Dark Mode & RTL Support**: Built-in, fully functional toggles for Light/Dark themes and Right-to-Left layouts using pure CSS variables.
- **Responsive Design**: Mobile-first architecture ensures the site looks stunning on phones, tablets, and desktops.
- **Dashboards**: Fully stylized Admin and User dashboards with integrated Chart.js visualizations.
- **Custom CSS Architecture**: Vanilla CSS using extensive custom properties (CSS variables) for effortless customization.
- **No Dependencies**: Minimal external libraries. Uses only Phosphor Icons and Chart.js via CDN.
