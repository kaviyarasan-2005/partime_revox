# Changelog

All notable changes to the KILN template will be documented in this file.

## [1.0.0] - 2026-06-16

### Added
- Initial release of the KILN Pottery Studio HTML Template.
- Core CSS architecture (Variables, Utilities, Grid).
- Dynamic Theme Toggling (Light/Dark mode).
- Dynamic RTL Layout Support.
- Intersection Observer scroll animations (`.fade-in-up`).
- 14 distinct HTML pages created:
  - Home Variant A (`index.html`)
  - Home Variant B (`home-b.html`)
  - About (`about.html`)
  - Services (`services.html`)
  - Contact (`contact.html`)
  - Booking (`booking.html`)
  - Gallery (`gallery.html`)
  - Blog (`blog.html`)
  - Login (`login.html`)
  - Signup (`signup.html`)
  - Admin Dashboard (`admin-dashboard.html`)
  - User Dashboard (`user-dashboard.html`)
  - 404 Error (`404.html`)
  - Coming Soon (`coming-soon.html`)
- Chart.js integration for admin and user dashboards (`dashboard.js`).
- Documentation setup (`README.md`, `installation.md`, `customization.md`, `changelog.md`).
